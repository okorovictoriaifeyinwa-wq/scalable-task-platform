# Scalable Task Platform

A backend project built with Node.js and Express.

## 🚀 Features
- Basic server setup
- API-ready structure
- Scalable backend foundation

## 🛠 Tech Stack
- Node.js
- Express

## ▶️ How to Run

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
node index.js
```

3. Open in browser:
http://localhost:3000
